# Antigravity Astro Website

This is a premium Astro website template featuring:
- **Astro 5.0**
- **TailwindCSS** for styling
- **Glassmorphism** design with dark mode
- **Responsive** layout
- **Micro-animations** using pure CSS

## How to Run

1.  Ensure you have Node.js installed.
2.  Open a terminal in this directory.
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
5.  Open [http://localhost:4321](http://localhost:4321) in your browser.

## Project Structure

- `src/pages/index.astro`: Main landing page.
- `src/layouts/Layout.astro`: Base layout with dynamic background.
- `src/components/`: Reusable UI components.
